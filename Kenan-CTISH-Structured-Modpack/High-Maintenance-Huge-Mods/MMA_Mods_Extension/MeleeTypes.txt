	Weapon type classifications for use with martial arts.
	
	If a martial art uses 2 different items from one category, all items in that category can be added to it.
	
	Length is decided using longest side when possible. For our purposes, Man > 120cm, Arm = 70cm, Forearm (Including Hand) = 50 cm, Hand = 25 cm. Length can be approximate if longest side is not available.
	
	Omnidirectional means weapon can be held in any orientation and swung along any axis (clubs). Bidirectional means the weapon must be held in one of 2 possible orientations and swung along that axis (hammer).
	
	If an attribue is listed a weapon must have that attribute to be considered for that weapon category. If any attribute is not listed that attribute is optional. Length can be approximate.
	
	If a weapon part is listed the weapon must have that part. Parts are functional/visual and do not need to be sepearate pieces (ie a mace must have a shaft and head, but they can be from one piece of steel).
	
	A weapon can go in multiple categories if appropriate.
	
	If there is a good reason a weapon should or should not go in a category in spite of its requirements, please state why in ( ) under the category title.
	
	If a vanilla weapon is not used in any vanilla martial art or MMA it can be left out of this list, as it assumed that weapon is not suitable for martial arts.
